#### description
такой хеш-тег используется для того что бы определить причастность одного куска документа к какому-то документу отличному от текущего документа который содержит данный тег. Иными словами, этот тег создан для объединения фрагментов текста вырывая их из контекста.

такой хеш-тег тег можно объединять с другими хеш-тегами. Популярное сочетание это тот же одноименный тег + /head , например: #id12341212am121212123/head #id12341212am121212123 , таким образом сразу видно что данная запись является Оригинальной, или стартовой на которую ссылаются другие записи снабженные тегом #id12341212am121212123 без окончания /head или их еще можно назвать дочерние

так же удобно компоновать треды `#thread/head` в сочетании с #id12341212am121212123 или #id12341212am121212123/linkedlist/1 #id12341212am121212123/linkedlist/2 #id12341212am121212123/list/1 .
